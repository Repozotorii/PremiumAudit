АУДИТ

Внутренний аудит, включая операционный аудит, аудит договоров Проведение аудиторской проверки финансовой отчетности Проведение обзорных проверок бухгалтерии компании

Для инвестора:

1. Дает оценку рискам дальнейшего функционирования бизнеса;

2. Раскрывает слабые места, которые не упомянул продавец;

3. Защищает от потенциальных негативных финансовых событий.

Для менеджмента:

1. Оценивает работу штата сотрудников в бухгалтерии;

2. Внедряет схемы для налогового/финансового планирования;

3. Предоставляет рекомендации для минимизации фискальных, финансовых рисков;

4. Улучшает управление финансами;

5. Вносит ясность ведения бухгалтерии.

Для собственника:

1. Определяет достоверность данных, которые предоставляют подчиненные;

2. Позволяет проверить состояние документации на сегодняшний день;

3. Производит оценку стоимостии собственного бизнеса.
